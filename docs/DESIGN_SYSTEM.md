# Design System

## Direction

Professional, simple, fast, and readable. This is a financial decision tool, not a game.

## Layout

- Mobile-first
- Maximum content width: 1100px
- Comfortable touch targets
- Clear separation between inputs and results
- Results prioritize the decision and dollar amounts

## Components

- App header
- Category cards
- Calculator cards
- Labeled money input
- Labeled percentage input
- Results panel
- Recommendation badge
- Inline validation message
- Reset and calculate actions

## Accessibility

- All inputs require visible labels
- Do not rely on color alone
- Maintain visible keyboard focus
- Error messages must explain how to fix the input
- Respect reduced-motion preferences
